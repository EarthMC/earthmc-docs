# EarthMC Documentation

![hub](/img/emchub.png)

Welcome to EarhtMC, a geopolitical sandbox server for Minecraft Java edition. Create a town, join a nation and explore the custom earth map with your friends. The goal of the server's documentation is to help you getting started on the server and answer most questions you may have as a new player.
